# Weekly Step Tracker

This project is a Python program that records a user's daily step counts for one week and evaluates their performance against a daily step goal.

This was a class assignment for MIS 310 at CCSU.

## Project Overview
The program prompts the user to enter a daily step goal, records steps for seven days, calculates the average daily steps, and reports whether the goal was exceeded, met, or missed on average.

The project includes:
- A function to set the daily step goal
- A function to record seven days of step data
- A function to evaluate weekly performance
- Average step calculation
- Console-based user interaction

## Files
- `weekly_step_tracker.py` - Python script for the step tracking program

## Tools Used

## How to Run
1. Download or clone this repository.
2. Open the project folder in PyCharm or another Python IDE.
3. Run `weekly_step_tracker.py`.
4. Enter the requested values in the console.
